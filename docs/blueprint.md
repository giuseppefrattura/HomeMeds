# **App Name**: HomeMeds Manager

## Core Features:

- Manual Entry: Add medications manually by entering name, quantity, and expiration date.
- Medication List: Display medications in a list format, sorted by expiration date, and showing quantity available.
- Search and Filter: Filter and search medications by name.
- Expiry Reminders: Set reminders for medication expiration dates.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to evoke calmness and trust.
- Background color: Very light blue (#F0F8FF), nearly white, for a clean and organized feel.
- Accent color: Muted green (#90EE90) for positive notifications like successful entry or far-off expiry dates.
- Body and headline font: 'Inter', a grotesque-style sans-serif font for a clean and modern feel.
- Use simple, clear icons to represent medication types and actions.
- Clean, list-based layout for easy medication overview.
- Subtle transitions for adding and removing medications.
